####小遣い帳？(Allowance book...?)

収支を入力して残高を計算してくれるプログラム

#####実行例
```
/Users/rumagishi/scala% scala Kozukai
収支金額を入力してください．
0 : 現在の残高を表示します．
>>> -10000
15/04/10 1:58 120000
/Users/rumagishi/scala%
```
